Our Purpose

This capstone project was designed to bridge that gap by bringing formal education from two of the leading learning platforms:Amazon Web Services (AWS) and IBM SkillsBuild. Together with real-world implementation. The aim of the project was to apply basic cybersecurity principles such as identity and access management (IAM), data encryption, and network security in the development of a secure, cloud-based web application on AWS infrastructure. By translating theoretical ideas into an operational and user-understandable system, the project was to demonstrate technical ability, promote practical preparedness, and create industry-specific cybersecurity expertise.

Key Topics

Network Security

Protecting network infrastructure through firewalls, IDS/IPS, and secure protocols.

Encryption

Safeguarding data using modern cryptographic techniques and protocols.

Access Control

Managing authentication and authorization to protect sensitive resources.

Malware Protection

Defending against viruses, trojans, ransomware, and other malicious software.

Incident Response

Incident response handles, contains, and recovers from cyber threats.

Ethical Hacking

Ethical hacking tests systems for vulnerabilities to improve security.
